# Contribuer à OmniAgent

Nous sommes ravis que vous souhaitiez contribuer à **OmniAgent** ! Voici quelques directives pour nous aider à maintenir un code de qualité.

## 🚀 Comment contribuer ?

1.  **Signaler des bugs** : Utilisez les "Issues" de GitHub pour signaler tout comportement inattendu.
2.  **Proposer des fonctionnalités** : Vous avez une idée géniale (ex: un nouveau plugin) ? Ouvrez une Issue pour en discuter.
3.  **Améliorer le code** :
    - Clonez le dépôt.
    - Créez une branche thématique.
    - Assurez-vous que votre code suit le style du projet (Vanilla JS, CSS pur, Rust performant).
    - Soumettez une Pull Request (PR).

## 🛠️ Stack Technique

- **Frontend** : HTML5, CSS3 (Glassmorphism), JavaScript ES6+.
- **Backend** : Rust, Tauri v2.
- **Communication** : API REST & Streaming (SSE).

## 📝 Règles de conduite

- Soyez respectueux envers les autres contributeurs.
- Documentez vos changements dans la PR.
- Ne soumettez pas vos clés API personnelles dans vos commits !

Merci pour votre aide !
